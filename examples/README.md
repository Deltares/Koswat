# Koswat Examples

[![Python 3.13](https://img.shields.io/badge/Python-3.13-blue.svg)](https://www.python.org/downloads/release/python-3135/)
[![Binder branch](https://github.com/Deltares/koswat/actions/workflows/deploy_binder.yml/badge.svg)](https://github.com/Deltares/koswat/actions/workflows/deploy_binder.yml)
[![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/Deltares/koswat/jupyter-binder)

This directory contains all (current) available examples of koswat. An environment is ready to run them:
```
conda env create -f environment.yml
conda activate koswat_examples_env
python import koswat
```

At the same time you may use our [binder environment](https://mybinder.org/v2/gh/Deltares/koswat/jupyter-binder).

Please refer to our [Koswat documentation](https://deltares.github.io/koswat/) for more information.

## Binder requirements.
Binder requires the `environment.yml` file to be present. Do not remove them from this repository.
